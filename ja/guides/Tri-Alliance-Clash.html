トライ・アライアンス・クラッシュ 勝利ガイド

ひと目で分かるルール要約

参戦条件（要約）

  • 同盟戦力上位20位以内
  • 自動：第1軍団は基本解放
  • 第2軍団解放例：同盟Lv.6以上 + 累積接続人数40人
  • 軍団：参加者 + 候補者 合計15人以上で第2軍団を別計算可能
  • 個人：同盟加入 + 都市センターLv.16以上、5日以上未接続は参加制限の可能性

基本ルール（運用）

  • 投票 / 申請 / マッチング / 戦闘はサーバーカレンダーに従って進行
  • R4以上：申請および参加人数の調整が可能
  • 最大30人参加 + 候補者10人（サーバールールにより異なる）
  • 同盟ごとに最大2軍団で別戦場を同時進行可能
  • 各領主は合計3部隊を戦場に投入可能

要点 人員登録が勝敗に直結します。実際に参加できる人だけを登録してください。

戦闘段階（4ステージ）

段階 時間 核心目標
1) 準備 3分 役割分担、移動ルート確認、初動占領ルート整理
2) 占領拡張 17分 自陣先取り + 敵陣の小型建物を一部奪取（ガリソンは保護中）
3) ガリソン占領 20分 A24 / B24 / C24 解放。ガリソンは分間得点が高く勝負所
4) 天空攻勢 20分 40分で 潮の神殿（天空） 解放。初回占領 + 維持得点で最終決着

重要タイミング 20分（ガリソン） / 40分（天空）。このタイミングにエネルギーを残しておくことが勝利の鍵です。

事前準備（登録 / 人員 / 音声）

1) 登録

  • 実際に参加できる人だけを登録してください。幽霊登録はマッチング・実戦の両方で不利です。
  • R4以上の参加を推奨します。占領後に キャプテン 任命が可能で、エネルギー回復に有利です。

2) ボイスチャット

  • Discordボイス推奨です。最低でもリーダー1〜3人はリアルタイム指示ができる状態が理想です。
  • リアルタイムコールはチャットより圧倒的に速く、ガリソンや天空タイミングで差が出ます。

おすすめ編成

目標は、得点生産（占領 / 維持）と重要オブジェクト防衛を同時に回すことです。

攻撃隊

  • 敵陣進入 → 建物奪取 → 20分ガリソン攻略 → 40分天空ラッシュ
  • 最も強い部隊を優先配置

防衛隊 1 / 2

  • 自軍の重要ルートと中継拠点を防衛
  • 特に20分以降のガリソン死守が最優先

遊軍（フロート）

  • バックアップ / 増援担当、状況に応じて攻撃または防衛支援
  • エネルギーを温存し、急変した局面に反応

英雄 / 部隊選択

  • 各領主は3部隊（合計9英雄）を運用
  • 最強の1部隊をガリソン / 天空タイミング用に温存する運用も有効

ステージ別運用戦略

ステージ 1〜2（0〜20分）：準備 + 占領拡張

  • 開始直後に自陣建物を先取りし、中継拠点（Transit Hubs）を確保します。
  • 攻撃隊は敵陣の小型建物を一部奪い、得点と進軍ルートの優位を取ります。
  • 序盤目標は分間得点で優位を作ることです。
  • エネルギーの使い過ぎ禁止：20分タイミングに向けてエネルギー30〜40%を残す運用が理想です。

ステージ 3（20〜40分）：ガリソン

  • 20分でガリソン（A24 / B24 / C24）が解放されます。
  • 優先順位1：自軍ガリソン防衛。失うと点差が一気にひっくり返る可能性があります。
  • 優先順位2：可能であれば 敵ガリソン奪取 で差を広げます。
  • 遊軍は戦況が崩れ始めた地点へ即投入します。

ステージ 4（40〜60分）：天空（潮の神殿）

  • 40分で天空（潮の神殿）が解放されます。初回占領ポイント が大きいため、ラッシュのタイミングが重要です。
  • 攻撃隊は天空ラッシュ / 維持、防衛隊は進入路（A29 / B29 / C29）維持に集中します。
  • 遊軍は敵増援を遮断し、接続建物の奪還 / 再占領と天空防衛補強を担当します。

勝利公式 （ガリソン維持）+（天空初回占領）+（天空維持）+（進入路防衛）の4つを同時に満たせるほど勝率が上がります。

実戦Tips（エネルギー / 回復 / 再入場）

エネルギー管理

  • エネルギーは移動、占領、突撃、撤退、即時復活などに消費されます。
  • 占領後に キャプテン を任命すると、エネルギー回復面で有利になります。

再入場 / バフ

  • 戦場から離脱するとクールタイム後に再入場できます。例として12分ルールが使われる場合があります。
  • 行軍速度 / 収容量関連の一部を除き、多くのバフは有効で、最終基準はサーバールールに従います。

負傷 / 回復運用

  • 大きく削られたら一度後ろの建物へ下がって回復し、再入場する方が即時復活より損失が少ない場合が多いです。
  • 回復 / 徴兵は、交戦が一瞬切れるタイミングに合わせると効率が良くなります。

FAQ

Q. なぜ各チームの Ruin Clusters 数が違って見えるのですか？

マップバランスは、単純に同じ数へ揃えるよりも、ルート距離、中継拠点への到達性、ガリソン防衛難度、天空進入の安定性まで含めて設計されることが多いです。 つまり、Ruin Clusters が多く見える側は、そのぶん防衛負担が大きかったり、重要接続地点の管理難度が高かったりする場合があります。 実戦では、遺跡数より 20分ガリソン と 40分天空 の運用で勝負が決まることが多いです。

Q. このイベントでいちばん重要なことは？

エネルギーです。序盤でエネルギーを使い切ると、20分 / 40分の勝負所で何もできません。 占領後のキャプテン任命、役割分担、遊軍運用でエネルギーを残しておくことが勝利の核心です。
