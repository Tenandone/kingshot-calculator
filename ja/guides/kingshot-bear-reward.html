キングショット ジャイアントベア報酬まとめ

ジャイアントベアは48時間ごとに1回参加できるコンテンツです。 つまり2日に1回参加できるため、1か月間欠かさず進めると合計15回プレイできます。

今回の整理は 個人ダメージポイント175M以上 を基準に計算しており、 クラフトハンマー / 英雄装備箱 / 強化パーツ / 同盟コイン を中心に見やすくまとめました。

ひと目でわかるポイント

  • ジャイアントベアは 2日に1回 参加可能
  • 30日フル参加基準で 合計15回 進行可能
  • 低い区間でも積み重ねると報酬はかなり大きい
  • 表の開始区間は 個人ダメージポイント175M以上

ジャイアントベアは一般的なイベントのように毎日開く構造ではなく、 48時間に1回 参加できる周期型コンテンツです。

ポイント： 開始区間である175M以上を安定して維持するだけでも、 1か月基準で クラフトハンマー150個 / 英雄装備箱30個 / 強化パーツ165個 / 同盟コイン900K が累積します。

要約

  • ジャイアントベアは 48時間ごとに1回、つまり 2日に1回 参加可能
  • 30日基準で欠かさず参加すると 合計15回 進行可能
  • 今回の表は 個人ダメージポイント175M以上 を基準に計算
  • 主な報酬は クラフトハンマー / 英雄装備箱 / 強化パーツ / 同盟コイン
  • 低い区間でも継続して積み重ねれば体感できるほど報酬が大きい
ひとことでまとめると： ジャイアントベアは2日に1回のコンテンツなので、 1か月フル参加基準で合計15回、欠かさず参加するだけでも累計報酬はかなり大きくなります。
