킹샷 서버 성장 타임라인 가이드

영주 여러분, 아래는 킹샷 서버 기준 주요 콘텐츠 오픈 일정입니다. 1세대~6세대 영웅과 펫, 순금 콘텐츠, 순금용광로, 전쟁아카데미, 서버전(SvS)까지 서버 성장 흐름에 맞춰 한 번에 정리했습니다.

실제 적용 일정은 서버 진행도에 따라 다소 차이가 있을 수 있습니다. 보통 최소 10일에서 최대 20일 정도 차이가 발생할 수 있으므로 참고용으로 활용해 주세요.
개요• 타임라인• 관련 링크• 주의사항

개요

이 페이지는 킹샷 서버가 성장하면서 어떤 콘텐츠가 언제쯤 열리는지 빠르게 확인할 수 있도록 만든 타임라인 가이드입니다. 특히 영웅 세대 교체, 펫 출시, 순금 단계, 순금용광로, 전쟁아카데미, 서버전(SvS) 시점 같은 핵심 흐름을 한눈에 파악할 수 있습니다.

각 일정은 서버 생성일 기준 예상 일차이며, 실제 서버 상태에 따라 오픈 시점은 달라질 수 있습니다. 따라서 절대적인 고정 일정이라기보다 준비 타이밍을 잡는 참고용으로 보는 것이 좋습니다.

서버 생성일 확인 팁: VIP를 누락 없이 진행했다면 해당 기록 기준으로 확인할 수 있고, 만약 누락이 발생했다면 이정표 최상단의 목표 달성일에서 7일을 빼면 서버 생성일입니다.
본 포스팅은 서버 진행 상황에 따라 지속적으로 보완되며, 최종적으로 TG10 구간까지 업데이트될 예정입니다.
