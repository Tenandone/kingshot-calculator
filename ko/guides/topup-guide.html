모바일 게임 유저들을 위한 합리적인 충전 가이드

[루트바 가입 링크] 로 가입하세요.
검색창에 #s4qU 입력 시 추가 쿠폰을 받을 수 있습니다.

1. 단품 충전

충전하고 싶은 패키지를 선택한 뒤 Top-up Now를 누릅니다.

2. 다품 충전

장바구니 버튼으로 원하는 패키지를 여러 개 추가합니다.

3. 결제 진행

결제 수단을 카드 또는 PayPal 중 선택한 뒤 Pay Now를 누릅니다.

좌측은 셀프탑업(내가 직접 충전), 우측은 상담원 충전입니다. 셀프탑업을 선택하세요.

4. 로그인

셀프탑업 선택 시 로그인 화면이 표시됩니다. 구글 로그인이 가장 편리합니다.

  • 구글 이메일 입력
  • 비밀번호 입력

※ 초반에는 비밀번호를 미리 정확히 확인 후 진행하세요.

5. 장바구니 충전

장바구니에 담긴 금액(합계)만큼 충전합니다.

6. 구매 완료

확인 후 결제가 진행되며, 완료 화면에서 Got it을 누르면 종료됩니다.

충전 완료!

최종 완료

모든 과정이 완료되었습니다. 게임에서 충전된 패키지를 확인하세요.

셀프충전은 매우 간단합니다.
문의가 많아 본 가이드로 안내를 대신합니다.
[누적충전 바로가기 링크] 를 즐겨찾기(북마크) 해두세요.
매월 말 집계하여 $4.99~$99.99 패키지를 추가로 증정합니다.
