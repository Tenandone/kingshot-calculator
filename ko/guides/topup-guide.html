킹샷 셀프충전 가이드

막히는 구간만 먼저 확인하면 셀프충전은 생각보다 간단합니다.

계정 연동 조건, 구글/페이스북 2차인증 확인 방법, 구글플레이게임즈 연동 불가 안내, 실제 충전 진행 순서를 한 번에 정리했습니다.

셀프충전이 막히는 대부분의 원인은 충전 방법 자체보다 계정 연동 방식과 2차인증 설정 여부입니다.

루트바가 인게임 충전보다 더 저렴하다면 같은 예산으로도 더 효율적으로 성장할 수 있습니다. 계정 준비만 끝났다면 아래 페이지에서 바로 셀프충전을 시작하면 됩니다.

필수 조건만 한 번 확인하고, 더 효율적인 충전을 가장 빠르게 진행해보세요.

커뮤니티에서 자주 막히는 부분 기준 정리 막히면 디스코드에서 안내 가능

셀프충전 전 꼭 먼저 확인해주세요

중요: 본인의 개인정보와 계정 보안은 본인이 직접 관리해야 합니다. 셀프충전을 진행하기 전 아래 2가지를 먼저 확인해주세요.
  • 킹샷 계정이 구글메일 또는 페이스북에 연동되어 있어야 합니다.
  • 연동된 구글메일 또는 페이스북 계정에 2차인증이 설정되어 있어야 합니다.

이 내용을 먼저 안내하는 이유는, 셀프충전이 막히는 분들 대부분이 충전 방법 자체보다 계정 연동 상태 또는 2차인증 미설정 때문에 먼저 막히기 때문입니다.

계정 연동 조건

루트바 셀프충전은 구글메일 또는 페이스북에 직접 연동된 계정만 이용 가능합니다.

Q. 저는 구글플레이게임즈로 연동되어 있습니다.

네. 루트바 이용 시 구글플레이게임즈로 연동된 계정은 셀프충전이 불가능합니다. 현재는 구글메일, 페이스북으로 연동된 계정만 셀프충전이 가능합니다.

구글메일 / 페이스북 2차인증이 꼭 필요한 이유

루트바 셀프충전은 로그인 보안 확인 과정이 필요합니다.

그래서 연동된 구글메일 또는 페이스북 계정에 2차인증이 설정되어 있어야 정상적으로 진행할 수 있습니다.

구글메일 2차인증 설정 방법

  • 구글 계정 관리 페이지에 접속합니다.
  • 보안 메뉴로 이동합니다.
  • 2단계 인증 항목을 찾습니다.
  • 휴대폰 번호, 인증 앱 등 원하는 인증 수단을 등록합니다.
  • 설정 완료 후 정상적으로 적용되었는지 확인합니다.

페이스북 2차인증 설정 방법

  • 페이스북 설정 및 개인정보 메뉴로 들어갑니다.
  • 비밀번호 및 보안 메뉴로 이동합니다.
  • 2단계 인증 사용 항목을 선택합니다.
  • 휴대폰 문자, 인증 앱 등 원하는 인증 방식을 등록합니다.
  • 설정 완료 후 정상적으로 적용되었는지 확인합니다.

2차인증이 설정되었는지 확인하는 방법

가장 쉬운 방법은 시크릿모드에서 직접 로그인해보는 것입니다.

  • 크롬 또는 사용 중인 브라우저에서 시크릿모드를 엽니다.
  • 구글 로그인 페이지 또는 페이스북 로그인 페이지로 이동합니다.
  • 본인 계정으로 로그인을 시도합니다.
  • 로그인 과정에서 휴대폰, 이메일, 또는 인증 앱으로 추가 인증 요청이 오면 해당 계정은 2차인증이 설정된 상태입니다.
확인 기준: 휴대폰 인증, 이메일 인증, 인증 앱 승인 요청 중 하나라도 나오면 2차인증이 적용된 것입니다.

1. 단품 충전

3. 결제 진행

좌측은 셀프탑업, 우측은 상담원 충전입니다. 셀프탑업을 선택하세요.

4. 로그인

셀프탑업 선택 시 로그인 화면이 표시됩니다. 구글 로그인이 가장 편리합니다.

  • 구글 이메일 입력
  • 비밀번호 입력
안내: 초반에는 비밀번호를 미리 정확히 확인 후 진행하세요.

5. 장바구니 충전

장바구니에 담긴 금액(합계)만큼 충전합니다.

6. 구매 완료

확인 후 결제가 진행되며, 완료 화면에서 Got it을 누르면 종료됩니다.
