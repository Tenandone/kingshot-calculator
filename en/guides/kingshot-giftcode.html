How to Redeem

  1. Go to Profile → Settings → Coupon Exchange in-game, or open the official redeem page.
  2. Enter the coupon code exactly.
  3. Claim your rewards from your mailbox.

Tip: Use the copy button first, then paste the code into the redeem field for convenience.

Quick Fix When a Code Does Not Work

Checking the items below in order solves most cases.

  • Enter the code exactly as shown and check uppercase and lowercase letters.
  • Check whether the code has expired based on UTC.
  • Check whether it is limited to one use per account.
  • Some codes also require a Town Center level condition.

Frequently Asked Questions

Are there any permanent codes?
This page only tracks clearly verified active codes and expired event code records. Codes with unclear status are not included in the active list.
The code is not working.
Make sure you entered the code exactly and check uppercase and lowercase letters. Also confirm expiration (UTC), one-use-per-account limits, and Town Center level requirements.
How often are new gift codes released?
They are released from time to time, and this page is updated as soon as a code is confirmed. Bookmark this page to check the latest codes quickly.
Why keep expired event codes listed?
They are kept as a record of codes distributed during past events. Any reissue is only confirmed through official announcements, while the archive section remains for search and reference purposes.
