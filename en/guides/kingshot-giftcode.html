How to Redeem Kingshot Gift Codes

  1. Open Profile → Settings → Redeem Code in the game, or go directly to the official redeem page.
  2. Enter the Kingshot code exactly as shown.
  3. Claim the rewards from your in-game mailbox.

On iOS, code redemption is not available inside the game, so please use the official redeem page.

What to Check if a Kingshot Code Does Not Work

Go through these checks one by one.

  • Enter the Kingshot code exactly as shown and double-check the letter case.
  • Check whether the code has expired using UTC time.
  • Some codes can only be redeemed once per account.
  • Some Kingshot codes require a minimum Town Center level.

Kingshot Gift Code FAQ

Where do I enter Kingshot gift codes?
You can enter them through Profile → Settings → Redeem Code in the game, or by using the official redeem page.
How do I redeem Kingshot codes on iOS?
On iOS, you cannot redeem codes inside the game. Please use the official redeem page instead.
Why is my Kingshot code not working?
Make sure you entered the code exactly as shown and check the letter case. Also confirm the expiration time in UTC, the one-time-per-account limit, and any Town Center level requirement.
Why keep expired codes on the page?
They are kept for reference. We update confirmed 2026 Kingshot codes as soon as they are verified, so it is worth checking back often.
