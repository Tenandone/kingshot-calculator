기본 정보

  • 개인 콘텐츠, 2주 주기 / 30분 진행, 자유 시간 선택 가능
  • 목표: 청동석 최대 채집 → 영주 보석 성장 핵심 재화
  • 광맥(1~3레벨): 레벨↑ = 청동석↑ / 수비 강함↑ → 전투력 맞춰 도전
  • 점령 시 손모양 청동석 드랍: 누구나 줍기 가능 → 즉시 획득

기본 전략

  • 강한 유저: 3레벨 광맥 위주 자리 → 수비병 수시 처치로 스킬 경험치 확보
  • 약한 유저: 2레벨 광맥 6채집 루틴 운영(안정 점수)
  • 체사레 수비병: 처치 시 스킬 경험치, 패배 시 해당 영웅 5분 사용 불가
시작 3분 대기 동안 2~3렙 광맥 + 수비병 밀집 지역을 파악한 뒤 도시 이전으로 유리한 위치 선점.

궁전 대응 전략

  • 강한 유저: 남은 7분 시점 중앙 궁전 진입 준비
  • 행군 가속 50% 3~4개 사용 시 약 2초 남기고 도착 가능(맵·거리 따라 변동)
  • 약한 유저: 궁전 경쟁은 피하고 광맥 채집에 집중 → 안정적 점수

스킬 트리 & 타이밍

스킬 빌드

오 - 오 - 왼 - 왼 - 오

  1. Lv1 (오): 수비병 처치 시 청동석 추가
  2. Lv2 (오): 부대 이동속도 +25%
  3. Lv3 (왼): 점령 성공 시 청동석 추가 (쿨타임 1분)
  4. Lv4 (왼): 채집 효율 +15%
  5. Lv5 (오): 채집 속도 +50% (60초, 쿨 5분)
  • 스킬 타이밍: 20분 / 15분 / 10분 / 5분에 한 번씩 사용
  • 특히 15분 타이밍만 맞춰도 1등 가능 → 쿨 누수 방지
  • 목표: 20분 전까지 Lv5 달성(수비병 처치로 경험치 수급)

불안정 광맥 타이밍

  • 채집 속도 빠름 → 항상 우선 점령 대상
  • 등장 시간: 남은 20·19·18분 / 10·9·8분 구간
  • 등장 알림 시 전 부대 투입, 손모양 드랍 즉시 줍기

핵심 팁

  • 광맥·수비병·줍기 루프 최적화: 이동 경로를 원형 루틴으로 설계
  • 패배 리스크 관리: 약한 영웅은 수비병 교전 지양, 채집/줍기에 집중
  • 경쟁 과열 지역 회피: 인접 대체 광맥으로 즉시 이동
패배 시 해당 영웅 5분 사용 불가 → 핵심 조합은 무리한 교전 금지

요약

  • 강한 유저: 3렙 광맥 + 수비병 + 7분 궁전 진입
  • 약한 유저: 2렙 광맥 6채집 루틴 + 수비병 중심
  • 스킬 타이밍: 20/15/10/5분
  • 체사레 수비병 처리 → 빠른 스킬 5렙 달성
  • “오-오-왼-왼-오”만 외워도 상위권 충분
