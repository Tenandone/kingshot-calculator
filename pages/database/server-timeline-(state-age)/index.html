서버 타임라인

개요

킹샷에서 왕국/서버는 시간이 지나감에 따라 새로운 기능과 영웅이 순차적으로 해금됩니다.
아래 타임라인은 대략적인 기준입니다.

서버 나이 계산 방법

1) VIP 로그인 일수로 추정

VIP 탭에서 누적 로그인 일수를 확인합니다. 서버 생성일에 맞춰 시작했고 매일 접속했다면,
이 값이 왕국/서버 생성 이후 경과 일수의 비교적 정확한 추정치입니다.

2) 업적 ‘별의 불꽃’으로 추정

도시 중앙에 있는 이정표로 가거나, 이벤트 페이지에서 접근합니다.
위에서 두 번째 업적인 ‘별의 불꽃’을 찾습니다. 이 업적은 보통 서버 생성 후 6일째 완료되며,
표시된 완료 날짜에서 6일을 빼면, 해당 왕국/서버의 생성일을 대략 추정할 수 있습니다.
