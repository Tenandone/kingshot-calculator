개요

킹샷에서 왕국/서버는 시간이 지날수록 새로운 기능과 영웅이 순차적으로 해금됩니다. 아래 타임라인은 대략적인 기준입니다.

서버 나이 타임라인

  • 0일→ 1세대 영웅 출시
  • 14일→ 평원 지역 안개 해제
  • 38일→ 옥토 지역 안개 해제
  • 40일→ 2세대 영웅 출시
  • 45일→ 연맹 자원 교환 기능 오픈
  • 54일→ 첫 캐슬 전투
  • 55일→ 1세대 펫 출시
  • 60~70일→ 순금 콘텐츠 시작 (순금 Lv.1~3)
  • 75~80일→ 2세대 펫 출시
  • 80일→ 최강왕국(SvS) / 지고의 영주 참가 가능
  • 100~110일→ 3세대 펫 출시
  • 100~120일→ 3세대 영웅 출시
  • 150일→ 순금 Lv.4~5 & 황금 용광로 오픈
  • 190~200일→ 4세대 펫 & 4세대 영웅 출시
  • 220~230일→ 전쟁 아카데미 (순금 연구소) 오픈
  • 260~280일→ 5세대 영웅 & 5세대 펫 출시
  • 310~320일→ 순금 Lv.8 해금
  • 360일+→ 6세대 영웅 출시 (업데이트 일정 미정)

서버 나이 계산 방법

1) VIP 로그인 일수로 추정

VIP 탭에서 누적 로그인 일수를 확인합니다. 서버 생성일에 맞춰 시작했고 매일 접속했다면, 이 값이 왕국/서버 생성 이후 경과 일수의 비교적 정확한 추정치입니다.

2) 업적 ‘별의 불꽃’으로 추정

도시 중앙에 있는 이정표로 가거나, 이벤트 페이지에서 접근합니다. 위에서 두 번째 업적인 ‘별의 불꽃’을 찾습니다. 이 업적은 보통 서버 생성 후 6일째 완료되며, 표시된 완료 날짜에서 6일을 빼면 해당 왕국/서버의 생성일을 대략 추정할 수 있습니다.
