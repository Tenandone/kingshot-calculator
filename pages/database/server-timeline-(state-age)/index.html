개요

킹샷에서 왕국/서버는 시간이 지날수록 새로운 기능과 영웅이 순차적으로 해금됩니다. 아래 타임라인은 대략적인 기준입니다.

서버 나이 타임라인

  • 0일→ 1세대 영웅
  • 14일→ 평원 지역 안개 해제
  • 38일→ 옥토 안개 해제
  • 40~50일→ 2세대 영웅
  • 45일→ 연맹 자원 교환
  • 54일→ 첫 캐슬 전투
  • 55~60일→ 1세대 펫
  • 70일→ 순금의 시대 (황금시대)
  • 75~80일→ 2세대 펫
  • 70~80일→ 왕국이 최강왕국 (서버전) 참가 자격 충족
  • 105~110일→ 3세대 펫
  • 105~120일→ 3세대 영웅
  • 150일→ 순금 5레벨 및 황금 용광로
  • 170~180일→ 4세대 영웅

서버 나이 계산 방법

1) VIP 로그인 일수로 추정

VIP 탭에서 누적 로그인 일수를 확인합니다. 서버 생성일에 맞춰 시작했고 매일 접속했다면, 이 값이 왕국/서버 생성 이후 경과 일수의 비교적 정확한 추정치입니다.

2) 업적 ‘별의 불꽃’으로 추정

도시 중앙에 있는 이정표로 가거나, 이벤트 페이지에서 접근합니다. 위에서 두 번째 업적인 ‘별의 불꽃’을 찾습니다. 이 업적은 보통 서버 생성 후 6일째 완료되며, 표시된 완료 날짜에서 6일을 빼면 해당 왕국/서버의 생성일을 대략 추정할 수 있습니다.
